# 技能自动评估决策树 (Windsurf Skill Decision Tree)

> **用途**: 作为独立规则文件供其他规则用 `@` 引用，决定技能自动路由。

```
START
│
├─ 是否需要“评估/可行性/方案/是否要做”？
│    └─ YES → @skills/feature-evaluation
│
├─ 是否涉及“权限/角色/访问控制/RBAC/守卫/数据范围”？
│    └─ YES → @skills/rbac
│
├─ 是否涉及“登录/注册/Auth.js/Session/OAuth/身份认证”？
│    └─ YES → @skills/auth
│
├─ 是否涉及“数据库/Prisma/迁移/Schema/Repository/SQL/查询”？
│    └─ YES → @skills/db-operations
│
├─ 是否涉及“构建/部署/打包/Turbopack/Standalone/Tauri/环境变量”？
│    └─ YES → @skills/build-standards
│
├─ 是否涉及“测试/Vitest/Playwright/单元/集成/E2E”？
│    └─ YES → @skills/testing
│
├─ 是否涉及“新技能/技能规范/技能模板/扩展技能”？
│    └─ YES → @skills/skill-creator
│
├─ 是否涉及“UI/页面/组件/表单/表格/弹窗/布局/交互/样式”？
│    ├─ 是否明确需要设计/配色/排版/视觉/动效？
│    │    └─ YES → @skills/ui-ux-pro-max
│    └─ NO  → @skills/ui-components
│
└─ 无法判断 → @skills/feature-evaluation
```
