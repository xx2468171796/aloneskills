## 对比度要求

WCAG AA 标准：
- 正常文本: ≥4.5:1
- 大文本: ≥3:1

| 背景 | 文本 | 对比度 |
|------|------|--------|
| `white` | `slate-900` | 15.8:1 ✅ |
| `white` | `slate-600` | 5.5:1 ✅ |
| `slate-900` | `slate-100` | 13.5:1 ✅ |
